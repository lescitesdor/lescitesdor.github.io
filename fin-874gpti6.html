Les couleurs que nous avons sélectionné pour rappeler l'univers de l'informatique sont principalement le vert flashy sur fond noir, en référence au film Matrix des années 90. En détail nous avons :
  • Pour le fond : un noir franc et plein, dont le code couleur est #000000
  • Pour les titres : un vert pétant et moche de code #13f917
  • Pour le texte : un vert plus doux pour le repos des yeux, avec le code #8af26a
  • Pour les mots importants : un jaune bien acidulé, de code #e9ff00
Les polices utilisées sont :
  • Tahoma : pour les textes
  • Georgia : pour les titres
  • Courier New, gras : pour les éléments mis en valeur
