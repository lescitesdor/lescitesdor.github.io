Accueil Parcours Géocaches

Ici un texte d'introduction sur le parcours. L'occasion de proposer une thématique, une histoire, une mise en scène qui personnalisera votre parcours de géocache.

Principe de fonctionnement

Le parcours Géocache est un jeu de piste qui conduit les joueurs à trouver des emplacements géographiques. Dans notre cas il s'agira d'emplacements dans le lycée.

Chaque emplacement est associé a une page web qui donne les informations de la prochaine cache :

  • Un indice sur l'emplacement exact. (texte, photo, devinette, image...)
  • Les coordonnées géographiques.

Ce que vous devez faire :

Chacun des membres de l'équipe doit réaliser les éléments suivants :

  • Une page web contenant :
    • Un texte donnant des indices sur l'emplacement du prochain QR Code
    • Une image (en rapport ou non avec l'incide).
    • Les coordonnées géographiques de l'emplacement.
    • Une mini-carte Google Maps (ou autre) représentant l'emplacement donné.

  • Générer un QR Code contenant l'URL de cette page web.

  • Remplir les fonctions du rôle qui vous est assigné (voir fiche rôle).

Pourquoi ?

Ce projet sera l'occasion pour nous d'expérimenter et aboder les notions suivantes :

  • Fonctionnement des URL.
  • Structuration des sites Web.
  • Accessibilité des sites Web, réseau, adresses.
  • Localisation géographique, longitude/latitude
  • Représentation numérique de la couleur.

Exemples de page à réaliser :

Exemple de page d'accueil
La page d'accueil doit faire la présentation de votre Ligne Editoriale : c'est à dire du thème/sujet traité ainsi que du ton employé.

Exemple d'indice
Une page indice est une page web qui donne des indications aux joueurs pour leur permettre de trouver le QR Code suivant.

Exemple de page de félicitations
La page de félicitation est la dernière page du parcours. Cette page contiendra un message de félicitation, ainsi qu'une présentation de la Charte Graphique utilisée sur le site.
